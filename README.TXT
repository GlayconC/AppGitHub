Desafios e Problemas:

 - Meu maior desafio foi que eu somente conhecia o REACT.
 - No meu trabalho atual utilizei bastante do React.js com JavaScript e as bibliotecas do MUI.
 - Mas pelo meu conhecimento anterior não tive muitos problemas e não encontrei dificuldades, somente levei um pouco mais de tempo até me adaptar as ferramentas.


Maneiras Para Melhorar o Código:

- Acredito que é possível diminuir a quantidade de linhas e ajustar as chamadas de API para maneiras mais favoráveis.
- Um exemplo, seria criar um arquivo.env para o token do github que foi necessário colocar pois ao recarregar várias vezes as páginas o github não aceitava mais requisições.
    - No código atual não possui o token.
- Não foi solicitado nos exemplos, mas poderia ser implementado um Search de usuário.

Instruções Para Rodar Aplicação

- Instalar o Node 
   - Comando: node -v

- Instalar o NPM ou Yarn
   - Comando NPM:npm -v
   - Comando YARN: npm install --global yarn

- Clonar o repositório
   - git clone https://github.com/GlayconC/AppGitHub
   - cd <nome-do-diretório>

- Instalar Dependências
   - npm install
   - yarn install (caso utilize o yarn)
